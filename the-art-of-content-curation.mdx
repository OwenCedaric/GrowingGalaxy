这是 `行内代码` 的示例。

### 代码块

使用三个反引号：

```
这是一个普通代码块
没有语法高亮
```

带语法高亮的代码块：

```python
def hello_world():
    print("Hello, World!")
    return True

# 这是 Python 代码
for i in range(10):
    hello_world()
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

// 这是 JavaScript 代码
greet("Markdown");
```

```html
<!DOCTYPE html>
<html>
<head>
    <title>示例</title>
</head>
<body>
    <h1>HTML 代码示例</h1>
</body>
</html>
```

## 数学公式

### 行内公式

这是一个行内公式示例：$$E = mc^2$$，爱因斯坦的质能方程。

圆的面积公式：$$A = \pi r^2$$

### 块级公式

**二次方程求根公式：**

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

**矩阵示例：**

$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$

**求和公式：**

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

**积分公式：**

$$
\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
